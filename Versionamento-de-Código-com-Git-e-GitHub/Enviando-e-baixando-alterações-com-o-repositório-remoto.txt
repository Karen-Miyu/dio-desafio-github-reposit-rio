#Enviando e baixando alterações com o repositório remoto
Código utilizado na aula:
````
git remote add origin 'URL'
git push -u origin main
#responsável por enviar o arquivo para o GitHub
#atualiza o seu github, e ele mostrará os arquivos salvos localmente.
git pull
#ele puxa as alterações remotas para localmente, mesclando com seu arquivo local.
````
